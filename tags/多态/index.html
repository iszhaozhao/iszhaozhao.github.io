C++学习笔记 - 多态和虚函数

2023-06-15
前言多态是面向对象编程的三大特性之一，也是 C++ 中最强大的特性之一。多态允许使用统一的接口来操作不同类型的对象，实现”一个接口，多种实现”。虚函数是实现多态的关键机制。 什么是多态多态是指相同的函数调用，根据对象类型的不同，表现出不同的行为。 C++ 中有两种多态： 类型 实现时机 例子 静态多态（编译时） 编译期确定 函数重载、运算符重载、模板 动态多态（运行时） 运行期确定...
